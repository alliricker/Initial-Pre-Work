/* CSS flexbox will fit all flex items together by default. Using the flex-wrap property will tell CSS to wrap the items. Extra items will move into a new row or column. The break point depends on the size of the container. CSS options include nowrap (default) wrap (wraps row items left to right and column items top to bottom) and wrap-reverse (wraps row items bottom to top and column items right to left). Adding the property flex-wrap to the #box-container element with the value of wrap will wrap the items to the next line. */
