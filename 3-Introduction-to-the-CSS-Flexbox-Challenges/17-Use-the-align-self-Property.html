/* The property align-self allows you to adjust each item's alignment individually. This is useful because other adjustment techniques such as float, clear, and vertical-align do not work on flex items. align-self will override any value set by the align-items property. Adding the align-self property to #box-1 with a value of center and #box-2 with a value of flex-end will center the blue box vertically and move the red box to the bottom of the column. */
