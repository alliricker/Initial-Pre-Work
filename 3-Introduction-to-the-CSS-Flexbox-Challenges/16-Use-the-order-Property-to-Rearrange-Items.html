/* The order property is used to tell CSS the order of how the flex items will appear in the flex container. The default is the order in which they are listed in the source HTML. The property uses numbers as values, including negative numbers. Giving #box-1 a value of 2 and #box-2 a value of 1 will swap the order of the boxes so that the red is displayed on the left and the blue is displayed on the right. */
